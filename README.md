## :books: 使用方法

### [1]源码运行(Linux/Arm64)

- 必要环境: **Python>=3.6** (Linux or Arm64)

命令示例:

    # ❗️桌面支持不全 或 字体符号不全 请关闭logo

    git clone https://github.com/cyear/ice-study.git

    cd ice-study

    pip3 install -r requirements.txt

    python3 main.py

### [2]直接运行(Windows)

- 运行环境: **Python>=3.6** (Windows>=7)

- Download exe

- Run exe
